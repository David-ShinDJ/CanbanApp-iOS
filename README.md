# SwiftUI 활용한 간단한 CanvanBoard 앱
## UI/UX 아래 이미지를 참조하였습니다
## 디자인패턴 칸반보드 특성상 보드마다 공유된 모델이 필요하는 함수가 비슷하여 MVC 패턴을 사용합니다
## Controller 각 뷰에서 공통되게 사용할수있게 SwiftUI의 StateObject 와 Published 사용하여 View Update 합니다
